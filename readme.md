### Simple Transit Pay

This application is mainly build for demo the transit payment 
cost of a 

pre-requisites

1. java 8+
2. maven
3. git

execution steps

1. clone the project
   
``` 
 git clone git-url
```

2. change directory in to simple-transit-pay

``` 
 cd simple-transit-pay
```

3. execute mvn clean install

``` 
 mvn clean install
```

4. run the application

``` 
 ./run.sh
```
